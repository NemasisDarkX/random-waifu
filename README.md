<p>Or You Can Deploy On Terminal
First Clone The Project</p>

```bash
$ git clone https://github.com/NemasisDarkX/random-waifu.git
```

<p>To Install The Packages</p>

```bash
$ npm i
```
<p>To Start The API</p>

```bash
$ npm test
```

<p>base url</p>

```bash
 $ http://localhost:3000/
 ```
<p>For Random Card</p>

```bash
$ http://localhost:3000/random/<waifu_name>
```
